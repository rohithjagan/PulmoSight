# Contributor Covenant Code of Conduct

## Our Pledge

We pledge to create an open, welcoming, and respectful environment for everyone.

---

## Expected Behavior

- Be respectful
- Be constructive
- Accept feedback gracefully
- Support collaborative learning
- Use inclusive language

---

## Unacceptable Behavior

- Harassment
- Discrimination
- Hate speech
- Trolling
- Personal attacks
- Sharing private information

---

## Enforcement

Project maintainers may remove or reject contributions that violate this Code of Conduct.

---

## Reporting

If you experience unacceptable behavior, contact the maintainers privately.
